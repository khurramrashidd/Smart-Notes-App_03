# 📝 React Notes App (Day 3)

A feature-rich **Notes Application built using React** that allows users to create, edit, search, and manage notes efficiently with persistent storage.

This project demonstrates real-world frontend concepts like **localStorage, dynamic rendering, and state management**.

This project is part of my **React learning journey (Day 3 Project)**.

---

## 🚀 Features

* Add new notes
* Edit existing notes
* Delete notes
* Search notes instantly
* Auto-save using localStorage
* Character counter while typing
* Dark / Light mode toggle 🌙
* Displays created date & time
* Press Enter to add notes
* Responsive card-based UI
* Sticky footer branding

---

## 🛠 Built With

* **React (Vite)**
* JavaScript (ES6+)
* CSS
* localStorage API

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/yourusername/day-03-notes-app.git
```

### 2. Navigate into the folder

```bash
cd day-03-notes-app
```

### 3. Install dependencies

```bash
npm install
```

---

## ▶️ Usage

Run the development server:

```bash
npm run dev
```

Open the local server URL in your browser.
